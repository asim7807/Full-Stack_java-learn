Installation & Setup

Follow these steps to install and set up Java on your machine...

Step 1: Check if Java is Installed

Open command prompt and type:

C:\Users\yourName> java -version

Output:

java version "1.8.0_281"
Java(TM) SE Runtime Environment
Java HotSpot(TM) 64-Bit Server VM

Step 2: Download Java

If your system does not have Java installed, download it from the official website: Download JDK,JVM & JRE

Step 3: Install Java

Run the installer and follow the steps.

Step 4: Set Up Environment Variables

  1. Right-click on This PC → Properties
  2. Advanced system settings → Environment Variables
  3. Edit PATH and add: C:\Program Files\Java\jdk1.8.0_xx\bin

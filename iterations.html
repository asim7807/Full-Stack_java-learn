Iteration Statements in Java

Introduction to Iteration Statements

Iteration statements, or loops, in Java allow a block of code to be executed repeatedly based on a condition. They are used to iterate over data or perform repetitive tasks efficiently.

Types of Iteration Statements

Java supports the following iteration statements:

  • for – Iterates a fixed number of times, typically used when the number of iterations is known.
  • while – Repeats as long as a condition is true, used when the number of iterations is unknown.
  • do-while – Similar to while, but guarantees at least one execution.
  • for-each – Iterates over arrays or collections, simplifying iteration over elements.

for Loop

The for loop is used when the number of iterations is known. It includes initialization, condition, and update steps.

public class ForLoopExample {
public static void main(String[] args) {
for (int i = 1; i <= 5; i++) {
System.out.println("Count: " + i);
}
}
}

Output

Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

while Loop

The while loop executes as long as a condition is true, suitable when the number of iterations is not predetermined.

public class WhileLoopExample {
public static void main(String[] args) {
int count = 1;
while (count <= 5) {
System.out.println("Count: " + count);
count++;
}
}
}

Output

Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

do-while Loop

The do-while loop executes at least once, then continues as long as the condition is true.

public class DoWhileLoopExample {
public static void main(String[] args) {
int count = 1;
do {
System.out.println("Count: " + count);
count++;
} while (count <= 5);
}
}

Output

Count: 1
Count: 2
Count: 3
Count: 4
Count: 5

for-each Loop

The for-each loop (enhanced for loop) simplifies iteration over arrays or collections.

public class ForEachLoopExample {
public static void main(String[] args) {
int[] numbers = {10, 20, 30, 40, 50};
for (int num : numbers) {
System.out.println("Number: " + num);
}
}
}

Output

Number: 10
Number: 20
Number: 30
Number: 40
Number: 50

Example Program with User Input

This program uses a while loop with user input to sum numbers until the user enters zero.

import java.util.Scanner;

public class IterationExample {
public static void main(String[] args) {
Scanner input = new Scanner(System.in);
int sum = 0;
int number;

System.out.println("Enter numbers to sum (enter 0 to stop):");
do {
System.out.print("Enter a number: ");
number = input.nextInt();
sum += number;
} while (number != 0);

System.out.println("Sum of entered numbers: " + sum);
input.close();
}
}

Output

Enter numbers to sum (enter 0 to stop):
Enter a number: 5
Enter a number: 10
Enter a number: 15
Enter a number: 0
Sum of entered numbers: 30

Important Notes

  • Use break to exit a loop prematurely.
  • Use continue to skip the current iteration and proceed to the next.
  • Avoid infinite loops by ensuring the loop condition will eventually be false.
  • The for-each loop is read-only for the iterated elements; it cannot modify the array or collection directly.

Java Math Class

Java's Math class provides methods to perform mathematical operations such as trigonometry, exponentiation, rounding, and more. It is part of the java.lang package and contains only static methods.

Math Constants

  • Math.PI = 3.1415926535 (value of π)
  • Math.E = 2.7182818284 (Euler's number)

Trigonometric Methods

All trigonometric methods use radians (not degrees). Use Math.toRadians() to convert degrees to radians.

MethodDescriptionExampleOutput
sin(x)Sine of angle (in radians)Math.sin(Math.PI / 2)1.0
cos(x)Cosine of angleMath.cos(0)1.0
tan(x)Tangent of angleMath.tan(Math.PI / 4)1.0
toRadians(x)Degrees → RadiansMath.toRadians(90)1.570...
toDegrees(x)Radians → DegreesMath.toDegrees(Math.PI)180.0

Example: Math Class Methods

                    
public class MathDemo {
  public static void main(String[] args) {
    System.out.println("PI: " + Math.PI);
    System.out.println("Max: " + Math.max(45, 78));
    System.out.println("Min: " + Math.min(45, 78));
    System.out.println("Sqrt: " + Math.sqrt(49));
    System.out.println("Power: " + Math.pow(2, 4));
    System.out.println("Random (1–100): " + (int)(Math.random() * 100 + 1));

    double angle = 90;
    double rad = Math.toRadians(angle);
    System.out.println("sin(90): " + Math.sin(rad));
    System.out.println("log(10): " + Math.log(10));
    System.out.println("log10(1000): " + Math.log10(1000));
  }
}

Output:

PI: 3.141592653589793
Max: 78
Min: 45
Sqrt: 7.0
Power: 16.0
Random (1–100): 27
sin(90): 1.0
log(10): 2.302585092994046
log10(1000): 3.0

Summary Notes

  • The Math class contains only static methods. No need to create an object.
  • Trigonometric methods use radians.
  • Math.random() gives random values between 0.0 and 1.0. Multiply & convert to int for ranges.
  • Use Math.toRadians(deg) for degree input in trig functions.

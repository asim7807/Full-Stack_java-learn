JDK, JRE, and JVM

The Java Development Kit (JDK), Java Runtime Environment (JRE), and Java Virtual Machine (JVM) are core components of the Java ecosystem.

Components of Java

  • JVM: Executes Java bytecode
  • JRE: JVM + libraries to run Java apps
  • JDK: JRE + tools to develop apps

Relationship Between JDK, JRE, JVM

How They Work Together

  1. Write Java source code
  2. Compile using javac
  3. Run with java command

Example: Java Program


// HelloWorld.java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

Steps:

  1. Compile: javac HelloWorld.java
  2. Run: java HelloWorld

Output:

Hello, Java!

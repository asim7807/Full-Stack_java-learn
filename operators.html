Java Operators

Operators in Java are special symbols that perform specific operations on one, two, or three operands and return a result. Java provides many types of operators to perform different operations like arithmetic, comparison, logic, bit manipulation, assignment, and more.

Why Operators Are Important?

  • They help manipulate variables and values easily.
  • They make mathematical, logical, and decision-based programming possible.
  • Operators are essential for control flow, loops, conditions, and expressions.

Types of Java Operators

  • Arithmetic Operators
  • Relational (Comparison) Operators
  • Logical Operators
  • Assignment Operators
  • Unary Operators
  • Bitwise Operators
  • Ternary Operator
  • Shift Operators

Arithmetic Operators

Used to perform basic arithmetic operations.

OperatorUseExampleResult
+Addition10 + 515
-Subtraction10 - 55
*Multiplication10 * 550
/Division10 / 25
%Modulus (remainder)10 % 31
                    
int a = 12, b = 5;
System.out.println("Add: " + (a + b));
System.out.println("Modulus: " + (a % b));

Output:

Add: 17
Modulus: 2

Relational (Comparison) Operators

Used to compare values and return boolean results.

OperatorMeaningExample
==Equal toa == b
!=Not equal toa != b
>Greater thana > b
<Less thana < b
>=Greater than or equala >= b
<=Less than or equala <= b
                    
int age = 18;
if (age >= 18) {
  System.out.println("Eligible to vote");
}

Output:

Eligible to vote

Logical Operators

Used for combining two or more conditions.

OperatorDescriptionExample
&&Logical AND(a > 5 && b < 10)
||Logical OR(a > 5 || b < 10)
!Logical NOT!(a > b)
                    
boolean hasLicense = true;
boolean hasID = false;
if (hasLicense && hasID) {
  System.out.println("Allowed to drive");
} else {
  System.out.println("Not allowed");
}

Output:

Not allowed

Assignment Operators

OperatorMeaningEquivalent To
=Assign valuex = 5
+=Add and assignx = x + 5
-=Subtract and assignx = x - 5
*=Multiply and assignx = x * 5
/=Divide and assignx = x / 5
%=Modulus and assignx = x % 5
                    
int num = 10;
num += 5; // num = 15
System.out.println(num);

Output:

15

Unary Operators

OperatorMeaningExample
++Incrementa++, ++a
--Decrementa--, --a
+Positive sign+a
-Negative sign-a
                    
int x = 5;
System.out.println(++x); // 6 (pre-increment)
System.out.println(x--); // 6, then x = 5

Output:

6
6

Bitwise Operators

OperatorDescription
&AND
|OR
^XOR
~Bitwise NOT
                    
int a = 5;  // 0101
int b = 3;  // 0011
System.out.println(a & b); // 0001 = 1
System.out.println(a | b); // 0111 = 7

Output:

1
7

Ternary Operator

Shorthand for if-else.

                    
int marks = 75;
String result = (marks >= 40) ? "Pass" : "Fail";
System.out.println(result);

Output:

Pass

Shift Operators

OperatorDescription
<<Left shift (multiply by 2ⁿ)
>>Right shift (divide by 2ⁿ)
                    
int x = 4;     // 0100
System.out.println(x << 1); // 8
System.out.println(x >> 1); // 2

Output:

8
2

Summary Notes

  • Use logical and relational operators in control statements.
  • Use bitwise operators for low-level manipulation or competitive coding.
  • Use parentheses to control precedence and avoid ambiguity.

Collections in Java

Introduction to Collections

The Java Collections Framework, part of the java.util package, provides a unified architecture for representing and manipulating groups of objects. Introduced in JDK 1.2, it includes core interfaces like Collection, List, Set, Queue, Deque, and Map, along with implementations such as ArrayList, LinkedList, HashSet, TreeSet, HashMap, and PriorityQueue. This framework enables dynamic data storage, efficient algorithms, and type safety through generics.

Detailed Examples of Collections

ArrayList Example: Student Grade Tracker

Manages a list of students with their grades, demonstrating add, remove, update, and iteration operations.

import java.util.ArrayList;

public class StudentGradeTracker {
public static void main(String[] args) {
ArrayList students = new ArrayList<>();
// Adding students
students.add("Alice - 85");
students.add("Bob - 92");
students.add("Charlie - 78");
students.add("David - 95");

System.out.println("Initial Student List: " + students);

// Update a grade
students.set(2, "Charlie - 88");
System.out.println("After updating Charlie's grade: " + students);

// Remove a student
students.remove("Bob - 92");
System.out.println("After removing Bob: " + students);

// Iterate using enhanced for loop
System.out.println("Student Grades:");
for (String student : students) {
System.out.println(student);
}

// Check size
System.out.println("Total Students: " + students.size());
}
}

Output

Initial Student List: [Alice - 85, Bob - 92, Charlie - 78, David - 95]
After updating Charlie's grade: [Alice - 85, Bob - 92, Charlie - 88, David - 95]
After removing Bob: [Alice - 85, Charlie - 88, David - 95]
Student Grades:
Alice - 85
Charlie - 88
David - 95
Total Students: 3

LinkedList Example: Inventory Management

Manages an inventory with add, remove, and peek operations at both ends.

import java.util.LinkedList;

public class InventoryManagement {
public static void main(String[] args) {
LinkedList inventory = new LinkedList<>();
// Adding items
inventory.add("Laptop");
inventory.addFirst("Mouse");
inventory.addLast("Keyboard");
inventory.add("Monitor");

System.out.println("Initial Inventory: " + inventory);

// Remove from start and end
String removedFirst = inventory.removeFirst();
String removedLast = inventory.removeLast();
System.out.println("Removed: " + removedFirst + ", " + removedLast);
System.out.println("After Removal: " + inventory);

// Peek at first item
System.out.println("First Item: " + inventory.peekFirst());

// Iterate
System.out.println("Remaining Items:");
for (String item : inventory) {
System.out.println(item);
}
}
}

Output

Initial Inventory: [Mouse, Laptop, Keyboard, Monitor]
Removed: Mouse, Monitor
After Removal: [Laptop, Keyboard]
First Item: Laptop
Remaining Items:
Laptop
Keyboard

TreeSet Example: Sorted Student Names

Maintains a sorted set of student names with automatic ordering.

import java.util.TreeSet;

public class SortedStudentNames {
public static void main(String[] args) {
TreeSet students = new TreeSet<>();
// Adding students
students.add("Charlie");
students.add("Alice");
students.add("Bob");
students.add("David");
students.add("Alice"); // Duplicate, ignored

System.out.println("Sorted Student Names: " + students);

// Remove a student
students.remove("Bob");
System.out.println("After removing Bob: " + students);

// First and last elements
System.out.println("First Student: " + students.first());
System.out.println("Last Student: " + students.last());

// Iterate
System.out.println("Students:");
for (String student : students) {
System.out.println(student);
}
}
}

Output

Sorted Student Names: [Alice, Bob, Charlie, David]
After removing Bob: [Alice, Charlie, David]
First Student: Alice
Last Student: David
Students:
Alice
Charlie
David

Queue Example: Task Scheduler with PriorityQueue

Manages tasks with priorities using a `PriorityQueue`.

import java.util.PriorityQueue;

public class TaskScheduler {
public static void main(String[] args) {
PriorityQueue tasks = new PriorityQueue<>();
// Adding tasks with priority (natural ordering)
tasks.add("Task3 - Priority 3");
tasks.add("Task1 - Priority 1");
tasks.add("Task2 - Priority 2");

System.out.println("Task Queue: " + tasks);

// Process tasks
while (!tasks.isEmpty()) {
System.out.println("Processing: " + tasks.poll());
}

// Try to poll from empty queue
System.out.println("Poll from empty: " + tasks.poll());
}
}

Output

Task Queue: [Task1 - Priority 1, Task3 - Priority 3, Task2 - Priority 2]
Processing: Task1 - Priority 1
Processing: Task2 - Priority 2
Processing: Task3 - Priority 3
Poll from empty: null

HashMap Example: Student Grade Map

Stores student names and grades in a `HashMap` with various operations.

import java.util.HashMap;

public class StudentGradeMap {
public static void main(String[] args) {
HashMap grades = new HashMap<>();
// Adding grades
grades.put("Alice", 85);
grades.put("Bob", 92);
grades.put("Charlie", 78);
grades.put("David", 95);

System.out.println("Initial Grades: " + grades);

// Update a grade
grades.put("Charlie", 88);
System.out.println("After updating Charlie: " + grades);

// Remove a student
grades.remove("Bob");
System.out.println("After removing Bob: " + grades);

// Check if key exists
System.out.println("Contains Alice: " + grades.containsKey("Alice"));

// Iterate over entries
System.out.println("Student Grades:");
for (String name : grades.keySet()) {
System.out.println(name + ": " + grades.get(name));
}
}
}

Output

Initial Grades: {Alice=85, Bob=92, Charlie=78, David=95}
After updating Charlie: {Alice=85, Bob=92, Charlie=88, David=95}
After removing Bob: {Alice=85, Charlie=88, David=95}
Contains Alice: true
Student Grades:
Alice: 85
Charlie: 88
David: 95

HashSet Example: Unique Product IDs

Maintains a set of unique product IDs with add, remove, and iteration.

import java.util.HashSet;

public class UniqueProductIDs {
public static void main(String[] args) {
HashSet productIDs = new HashSet<>();
// Adding product IDs
productIDs.add("P001");
productIDs.add("P002");
productIDs.add("P003");
productIDs.add("P001"); // Duplicate, ignored

System.out.println("Initial Product IDs: " + productIDs);

// Remove a product ID
productIDs.remove("P002");
System.out.println("After removing P002: " + productIDs);

// Check size and containment
System.out.println("Total Unique IDs: " + productIDs.size());
System.out.println("Contains P001: " + productIDs.contains("P001"));

// Iterate
System.out.println("Product IDs:");
for (String id : productIDs) {
System.out.println(id);
}
}
}

Output

Initial Product IDs: [P001, P002, P003]
After removing P002: [P001, P003]
Total Unique IDs: 2
Contains P001: true
Product IDs:
P001
P003

Core Interfaces and Implementations

  • Collection: Root interface.
  • List: ArrayList, LinkedList.
  • Set: HashSet, TreeSet.
  • Queue: PriorityQueue.
  • Map: HashMap.

Best Practices

  • Choose ArrayList for random access, LinkedList for frequent insertions.
  • Use TreeSet for sorted unique elements.
  • Employ PriorityQueue for prioritized tasks.
  • Utilize HashMap for key-value pairs, HashSet for uniqueness.

Important Notes

  • Part of java.util package.
  • Most collections are not thread-safe; use synchronized wrappers if needed.
  • Generics enhance type safety.

Java Versions

Java has evolved significantly since 1995. Below are major versions and features:

Major Java Versions

  • Java 8 (2014): Lambdas, streams, functional programming
  • Java 11 (2018): HTTP client, var keyword
  • Java 17 (2021): Sealed classes, enhanced switch
  • Java 21 (2023): Virtual threads, pattern matching

Example: Stream API


import java.util.Arrays;
import java.util.List;

public class StreamExample {
    public static void main(String[] args) {
        List numbers = Arrays.asList(1, 2, 3, 4, 5);
        numbers.stream()
               .filter(n -> n % 2 == 0)
               .forEach(System.out::println);
    }
}

Output:

2
4

Choosing a Java Version

  • Java 8: Widely used for legacy applications due to its stability and features like lambdas.
  • Java 11/17: Preferred for modern applications, offering long-term support and new features.
  • Java 21: Ideal for cutting-edge development with virtual threads for concurrency.
  • Check compatibility with your tools and libraries before upgrading.

Important Notes

  • LTS versions (8, 11, 17, 21) receive updates for several years, making them suitable for production.
  • Non-LTS versions (e.g., 9, 10, 12–16) are short-term and mainly for early adopters.
  • Use java -version to check your installed Java version.

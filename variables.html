Java Variables

A variable in Java is a memory location used to store data. It must be declared with a data type before it is used.

Syntax

                    
// General syntax for variable declaration
datatype variableName = value;

// Example:
int age = 20;
String name = "Asim";

Types of Variables in Java

  • Local Variable: Declared inside a method or block. Accessible only within that method or block.
  • Instance Variable: Declared inside a class but outside any method. Unique to each object.
  • Static Variable: Declared with the static keyword. Shared among all objects of the class.

Local Variable Example

                    
public class Test {
  void show() {
    // 'a' is a local variable declared inside the show() method
    int a = 10;
    System.out.println("Local variable a = " + a);
  }

  public static void main(String[] args) {
    // Create object of Test
    Test t = new Test();
    t.show(); // Call method to print local variable
  }
}

Output:

Local variable a = 10

Instance Variable Example

                    
public class Student {
  // Instance variable (each object gets its own copy)
  String name;

  // Method to set the name
  void setName(String n) {
    name = n; // Assign parameter to instance variable
  }

  // Method to display the name
  void display() {
    System.out.println("Student Name: " + name);
  }

  public static void main(String[] args) {
    Student s1 = new Student();  // Create first object
    Student s2 = new Student();  // Create second object

    s1.setName("Asim");
    s2.setName("Rahul");

    s1.display(); // Output: Student Name: Asim
    s2.display(); // Output: Student Name: Rahul
  }
}

Output:

Student Name: Asim
Student Name: Rahul

Static Variable Example

                    
public class College {
  // Static variable shared by all objects
  static String collegeName = "SIT Polytechnic";

  // Instance variable (unique to each object)
  String studentName;

  // Constructor to assign student name
  College(String name) {
    studentName = name;
  }

  void display() {
    // Accessing both static and instance variables
    System.out.println(studentName + " studies at " + collegeName);
  }

  public static void main(String[] args) {
    College c1 = new College("Asim");
    College c2 = new College("Rahul");

    c1.display(); // Output: Asim studies at SIT Polytechnic
    c2.display(); // Output: Rahul studies at SIT Polytechnic
  }
}

Output:

Asim studies at SIT Polytechnic
Rahul studies at SIT Polytechnic

Variable Types Summary Table

Type Declared In Scope Default Value
Local Method or block Within method/block only None (must be initialized)
Instance Inside class (outside methods) Accessible by object 0, null, false (depends on type)
Static Inside class using static Shared across all objects 0, null, false (depends on type)

Notes

  • Variables must begin with a letter, $, or _.
  • final keyword can be used to create constants.
  • Static variables can be accessed without creating objects.
  • Local variables do not get default values — must be initialized.

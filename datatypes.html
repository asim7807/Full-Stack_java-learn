Java Data Types

A data type in Java specifies the type and size of values that can be stored in a variable. Java is a strongly typed language, which means every variable must be declared with a data type before use.

Why are Data Types Important?

  • To define the nature of the data (e.g., number, character, boolean).
  • To ensure memory-efficient programming.
  • To avoid type mismatch errors at compile time.
  • To enable the compiler to allocate appropriate memory size.

Categories of Data Types

Java has two main categories of data types:

  • Primitive Data Types – Predefined, basic types
  • Non-Primitive Data Types – User-defined or derived types

Primitive Data Types

Java has 8 primitive types. These are the most basic data types and are not objects.

Data Type Description Size Default Value Example
byte Small integer 1 byte 0 byte b = 100;
short Medium integer 2 bytes 0 short s = 32000;
int Default integer type 4 bytes 0 int i = 50000;
long Large integer 8 bytes 0L long l = 10000000000L;
float Single precision decimal 4 bytes 0.0f float f = 3.14f;
double Double precision decimal 8 bytes 0.0d double d = 3.14159;
char Single character 2 bytes '' char c = 'A';
boolean True or false 1 bit false boolean isTrue = true;

Example: Using All Primitive Data Types

                    
public class PrimitiveExample {
  public static void main(String[] args) {
    byte age = 25;
    short year = 2025;
    int salary = 50000;
    long population = 8000000000L;

    float rating = 4.5f;
    double pi = 3.1415926535;

    char grade = 'A';
    boolean isJavaFun = true;

    System.out.println("Age: " + age);
    System.out.println("Year: " + year);
    System.out.println("Salary: " + salary);
    System.out.println("Population: " + population);
    System.out.println("Rating: " + rating);
    System.out.println("Pi: " + pi);
    System.out.println("Grade: " + grade);
    System.out.println("Is Java Fun? " + isJavaFun);
  }
}

Output:

Age: 25
Year: 2025
Salary: 50000
Population: 8000000000
Rating: 4.5
Pi: 3.1415926535
Grade: A
Is Java Fun? true

Non-Primitive Data Types

These data types are created by the user or provided by Java libraries. They store references (addresses) instead of actual values.

  • String – Sequence of characters
  • Array – Collection of elements of same data type
  • Class – Blueprint for creating objects
  • Interface – Abstract type for defining contracts
  • Objects – Instances of classes

Example: String and Array

                    
public class NonPrimitiveExample {
  public static void main(String[] args) {
    // String
    String name = "Asim";
    System.out.println("Name: " + name);

    // Array
    int[] marks = {85, 90, 95};
    for (int i = 0; i < marks.length; i++) {
      System.out.println("Mark " + (i + 1) + ": " + marks[i]);
    }
  }
}

Output:

Name: Asim
Mark 1: 85
Mark 2: 90
Mark 3: 95

Difference Between Primitive and Non-Primitive

Feature Primitive Non-Primitive
Type Basic types Reference types
Memory Stores actual value Stores address of object
Default Value Depends on type (0, false, etc.) null
Customizable? No Yes (created by programmer)

Important Notes

  • Use int for whole numbers, double for decimal numbers.
  • Always add L to long values and f to float literals.
  • char can store Unicode characters (like 'A', '$', '9').
  • String is not a primitive but widely used for text.

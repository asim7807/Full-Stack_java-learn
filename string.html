Java Strings

A String in Java is a sequence of characters. Strings are widely used for storing and manipulating text. Java provides the String class to work with strings.

Creating Strings

                    
// Using string literal
String name = "Asim";

// Using new keyword
String name2 = new String("Asim");

System.out.println(name);
System.out.println(name2);

Output:

Asim
Asim

Example: String Methods

                    
public class StringMethodsExample {
  public static void main(String[] args) {
    String str = "  Java Programming  ";

    System.out.println("Original String: \"" + str + "\"");

    // length()
    System.out.println("Length: " + str.length());

    // charAt()
    System.out.println("Character at index 2: " + str.charAt(2));

    // substring()
    System.out.println("Substring(2, 6): " + str.substring(2, 6));

    // toLowerCase()
    System.out.println("Lowercase: " + str.toLowerCase());

    // toUpperCase()
    System.out.println("Uppercase: " + str.toUpperCase());

    // equals()
    System.out.println("Equals 'Java Programming': " + str.trim().equals("Java Programming"));

    // compareTo()
    System.out.println("CompareTo 'Java': " + str.trim().compareTo("Java"));

    // trim()
    System.out.println("Trimmed: \"" + str.trim() + "\"");

    // contains()
    System.out.println("Contains 'gram': " + str.contains("gram"));

    // replace()
    System.out.println("Replace 'a' with '*': " + str.replace("a", "*"));

    // split()
    String[] words = str.trim().split(" ");
    System.out.println("Split by space:");
    for (String word : words) {
      System.out.println(word);
    }

    // indexOf()
    System.out.println("Index of 'a': " + str.indexOf("a"));

    // lastIndexOf()
    System.out.println("Last index of 'a': " + str.lastIndexOf("a"));

    // startsWith()
    System.out.println("Starts with '  Ja': " + str.startsWith("  Ja"));

    // endsWith()
    System.out.println("Ends with 'ing  ': " + str.endsWith("ing  "));

    // isEmpty()
    String empty = "";
    System.out.println("Is empty: " + empty.isEmpty());

    // equalsIgnoreCase()
    System.out.println("EqualsIgnoreCase: " + str.trim().equalsIgnoreCase("java programming"));

    // concat()
    String added = str.trim().concat(" Language");
    System.out.println("Concat: " + added);
  }
}

Important Notes

  • String methods are very useful in text processing, validation, and data parsing.
  • Strings are immutable. Use StringBuilder when modifying content frequently.
  • equals() vs ==: Use equals() for content comparison.

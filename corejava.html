Core Java Tutorial

Core Java refers to the fundamental aspects of the Java programming language used for general-purpose programming. It includes the basic libraries and tools that come with the Java Standard Edition (Java SE).

Why Learn Core Java?

  • Foundation for advanced Java (JEE, Spring, etc.)
  • Widely used in web, desktop, and mobile applications
  • Easy to learn with strong community support

Core Java Topics

  • Java Basics
  • OOPs Concepts
  • Data Types and Variables
  • Control Statements
  • Arrays
  • Strings
  • Exception Handling
  • Multithreading
  • File I/O
  • Collections Framework

Basic Structure of Java Program


public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, Java!");
  }
}

Key Features of Java

  • Object-Oriented: Everything in Java is an object.
  • Platform Independent: Write Once, Run Anywhere.
  • Robust: Java handles memory management and exceptions.
  • Secure: Provides runtime security and bytecode verification.
  • Multithreaded: Supports concurrent execution.

Java Variables Example


public class VariablesDemo {
  public static void main(String[] args) {
    int age = 25;
    String name = "Asim";
    System.out.println("Name: " + name);
    System.out.println("Age: " + age);
  }
}

Control Statements Example (if-else)


int number = 10;
if (number > 0) {
  System.out.println("Positive number");
} else {
  System.out.println("Negative number");
}

Loop Example (for loop)


for (int i = 1; i <= 5; i++) {
  System.out.println("Java is awesome!");
}

Object-Oriented Programming Example


class Student {
  int id;
  String name;

  void display() {
    System.out.println(id + " " + name);
  }

  public static void main(String[] args) {
    Student s1 = new Student();
    s1.id = 101;
    s1.name = "Asim";
    s1.display();
  }
}

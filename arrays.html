Arrays in Java

Introduction to Arrays

An array in Java is a data structure that stores a fixed-size sequence of elements of the same type. Arrays are used to store multiple values in a single variable, accessed by indices starting from 0.

Types of Arrays

Java supports two main types of arrays:

  • Single-dimensional arrays: A linear collection of elements of the same type.
  • Multi-dimensional arrays: Arrays of arrays, often used for matrices or tables (e.g., 2D arrays).

Operations on Single-Dimensional Arrays

Common operations include declaring, initializing, accessing, modifying, iterating, sorting, and searching.

1. Declaring and Initializing

Declare with the type followed by []. Initialize using new or an array initializer.

public class SingleArrayInit {
public static void main(String[] args) {
int[] numbers = new int[4]; // Declare array of size 4
numbers[0] = 10; numbers[1] = 20; numbers[2] = 30; numbers[3] = 40;
// Or use initializer
int[] numbers2 = {10, 20, 30, 40};
System.out.println("First element: " + numbers[0]);
}
}

Output

First element: 10

2. Accessing Elements

Use the index to access elements (0-based).

public class SingleArrayAccess {
public static void main(String[] args) {
int[] numbers = {10, 20, 30, 40};
System.out.println("Second element: " + numbers[1]);
}
}

Output

Second element: 20

3. Modifying Elements

Assign a new value to an index.

public class SingleArrayModify {
public static void main(String[] args) {
int[] numbers = {10, 20, 30, 40};
numbers[2] = 35; // Modify third element
System.out.println("Modified third element: " + numbers[2]);
}
}

Output

Modified third element: 35

4. Iterating

Use a for loop or for-each loop to iterate over elements.

public class SingleArrayIterate {
public static void main(String[] args) {
int[] numbers = {10, 20, 30, 40};
System.out.print("Elements (for loop): ");
for (int i = 0; i < numbers.length; i++) {
System.out.print(numbers[i] + " ");
}
System.out.print("\nElements (for-each): ");
for (int num : numbers) {
System.out.print(num + " ");
}
}
}

Output

Elements (for loop): 10 20 30 40
Elements (for-each): 10 20 30 40

5. Sorting

Use Arrays.sort() from java.util.Arrays to sort the array.

import java.util.Arrays;

public class SingleArraySort {
public static void main(String[] args) {
int[] numbers = {5, 2, 8, 1, 9};
Arrays.sort(numbers);
System.out.println("Sorted array: " + Arrays.toString(numbers));
}
}

Output

Sorted array: [1, 2, 5, 8, 9]

6. Searching

Use Arrays.binarySearch() for sorted arrays to find an element's index.

import java.util.Arrays;

public class SingleArraySearch {
public static void main(String[] args) {
int[] numbers = {5, 2, 8, 1, 9};
Arrays.sort(numbers); // Must sort before binary search
int index = Arrays.binarySearch(numbers, 8);
System.out.println("Index of 8: " + index);
}
}

Output

Index of 8: 3

Operations on Multi-Dimensional Arrays

Multi-dimensional arrays (e.g., 2D arrays) are arrays of arrays, often used for matrices. Operations are similar but require nested indices or loops.

1. Declaring and Initializing

Declare with multiple [], initialize with new or an initializer.

public class MultiArrayInit {
public static void main(String[] args) {
int[][] matrix = {
{1, 2, 3},
{4, 5, 6}
};
System.out.println("Element at [1][2]: " + matrix[1][2]);
}
}

Output

Element at [1][2]: 6

2. Accessing Elements

Use two indices to access elements in a 2D array.

public class MultiArrayAccess {
public static void main(String[] args) {
int[][] matrix = {{1, 2, 3}, {4, 5, 6}};
System.out.println("Element at [0][1]: " + matrix[0][1]);
}
}

Output

Element at [0][1]: 2

3. Modifying Elements

Assign a new value to a specific position using two indices.

public class MultiArrayModify {
public static void main(String[] args) {
int[][] matrix = {{1, 2, 3}, {4, 5, 6}};
matrix[0][1] = 20; // Modify element
System.out.println("Modified element at [0][1]: " + matrix[0][1]);
}
}

Output

Modified element at [0][1]: 20

4. Iterating

Use nested for loops or for-each loops to iterate over a 2D array.

public class MultiArrayIterate {
public static void main(String[] args) {
int[][] matrix = {{1, 2, 3}, {4, 5, 6}};
System.out.println("Matrix elements:");
for (int i = 0; i < matrix.length; i++) {
for (int j = 0; j < matrix[i].length; j++) {
System.out.print(matrix[i][j] + " ");
}
System.out.println();
}
}
}

Output

Matrix elements:
1 2 3
4 5 6

5. Sorting Rows

Sort individual rows of a 2D array using Arrays.sort().

import java.util.Arrays;

public class MultiArraySort {
public static void main(String[] args) {
int[][] matrix = {{3, 1, 2}, {6, 4, 5}};
System.out.println("Before sorting:");
for (int[] row : matrix) {
System.out.println(Arrays.toString(row));
}
for (int i = 0; i < matrix.length; i++) {
Arrays.sort(matrix[i]);
}
System.out.println("After sorting rows:");
for (int[] row : matrix) {
System.out.println(Arrays.toString(row));
}
}
}

Output

Before sorting:
[3, 1, 2]
[6, 4, 5]
After sorting rows:
[1, 2, 3]
[4, 5, 6]

6. Searching

Search for an element in a 2D array using nested loops or convert to a 1D array for Arrays.binarySearch().

public class MultiArraySearch {
public static void main(String[] args) {
int[][] matrix = {{1, 2, 3}, {4, 5, 6}};
int target = 5;
boolean found = false;
for (int i = 0; i < matrix.length; i++) {
for (int j = 0; j < matrix[i].length; j++) {
if (matrix[i][j] == target) {
System.out.println("Found " + target + " at [" + i + "][" + j + "]");
found = true;
break;
}
}
if (found) break;
}
}
}

Output

Found 5 at [1][1]

Example Program with User Input

This program allows users to input a 2D array and performs operations like printing and calculating the sum of elements.

import java.util.Scanner;

public class ArrayOperations {
public static void main(String[] args) {
Scanner input = new Scanner(System.in);
System.out.print("Enter number of rows: ");
int rows = input.nextInt();
System.out.print("Enter number of columns: ");
int cols = input.nextInt();
int[][] matrix = new int[rows][cols];

// Initializing
System.out.println("Enter matrix elements:");
for (int i = 0; i < rows; i++) {
for (int j = 0; j < cols; j++) {
System.out.print("Element [" + i + "][" + j + "]: ");
matrix[i][j] = input.nextInt();
}
}

// Printing
System.out.println("Matrix:");
for (int i = 0; i < rows; i++) {
for (int j = 0; j < cols; j++) {
System.out.print(matrix[i][j] + " ");
}
System.out.println();
}

// Calculating sum
int sum = 0;
for (int[] row : matrix) {
for (int num : row) {
sum += num;
}
}
System.out.println("Sum of elements: " + sum);
input.close();
}
}

Output

Enter number of rows: 2
Enter number of columns: 3
Enter matrix elements:
Element [0][0]: 1
Element [0][1]: 2
Element [0][2]: 3
Element [1][0]: 4
Element [1][1]: 5
Element [1][2]: 6
Matrix:
1 2 3
4 5 6
Sum of elements: 21

Important Notes

  • Array indices start at 0; accessing invalid indices causes ArrayIndexOutOfBoundsException.
  • Single-dimensional arrays are sorted using Arrays.sort(), but multi-dimensional arrays require sorting each row individually.
  • Multi-dimensional arrays can be jagged (rows of different lengths).
  • Use Arrays.binarySearch() only on sorted arrays for efficient searching.

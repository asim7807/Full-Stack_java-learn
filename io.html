User Input/Output

Taking Input:

To take input from the user in Java, we use the Scanner class which is part of the java.util package.

import java.util.Scanner;

We need to create a Scanner object and use its methods to read data from the user.

Example: Reading a String

Scanner input = new Scanner(System.in);
System.out.print("Enter your name: ");
String name = input.nextLine();
System.out.println("Hello, " + name + "!");

Example: Reading an Integer

Scanner input = new Scanner(System.in);
System.out.print("Enter your age: ");
int age = input.nextInt();
System.out.println("Your age is: " + age);

Example: Reading a Floating-point Number

Scanner input = new Scanner(System.in);
System.out.print("Enter your GPA: ");
float gpa = input.nextFloat();
System.out.println("Your GPA is: " + gpa);

Displaying Output:

To display output on the screen, we use:

  • System.out.print() – prints on the same line
  • System.out.println() – prints on a new line

Example:

System.out.print("Welcome ");
System.out.println("to Java!");

Important Notes:

  • Always close the Scanner object after use: input.close();
  • nextLine() reads a full line, including spaces.
  • next() only reads one word (stops at space).
  • Mixing nextInt() and nextLine() can cause issues; consume the leftover newline using input.nextLine() after nextInt().

Scanner Methods:

The Scanner class provides several methods to read different types of inputs from the user. Here's a list of commonly used ones:

Method Description Example
next() Reads a single word (no spaces). String name = input.next();
nextLine() Reads a whole line (including spaces). String line = input.nextLine();
nextInt() Reads an integer. int num = input.nextInt();
nextFloat() Reads a float value. float f = input.nextFloat();
nextDouble() Reads a double value. double d = input.nextDouble();
nextBoolean() Reads a boolean value (true/false). boolean flag = input.nextBoolean();
close() Closes the Scanner object. input.close();

Example Java Program

import java.util.Scanner;

public class UserInputExample {
  public static void main(String[] args) {
    Scanner input = new Scanner(System.in);

    System.out.print("Enter your name: ");
    String name = input.nextLine();

    System.out.print("Enter your age: ");
    int age = input.nextInt();

    System.out.print("Enter your percentage: ");
    float percent = input.nextFloat();

    System.out.println("\n--- Output ---");
    System.out.println("Name: " + name);
    System.out.println("Age: " + age);
    System.out.println("Percentage: " + percent + "%");
    input.close();
  }
}

Enter your percentage: 87.5
